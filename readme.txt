版本更新说明：
v0.1alpha,本人内测，构建雏形。
v0.2，增加32位与64位的版本切换功能。
v0.3，修改bt-trackers任务更新计划。
v0.4，优化ariaNG-Native快捷方式，在不对原版进行重新打包的情况下，可以在桌面自动生成超萌图标。
v0.5，优化重启服务，增加了对ariaNG-Native的重启支持。
v0.6，修复完全卸载时，无法停止所有服务的bug。
v0.7，修复bt-trackers自动更新过程中可能产生的bug。
v0.8，增加一键修改线程数功能。
v0.9，在一键改线程的同时，能自动更新bt-trackers。
v1.0，增加一键手动更新bt-trackers功能。
v1.1，增加彩蛋,并优化了图片字符画，修复小电视框问题。
v1.2，修复win7安装问题，并新增win7一键安装功能，同时与win10版做了区分。
v1.3，增加访问原帖功能。
v1.4，修复1903系统生成快捷方式出错的问题。
v1.5，增加ariaNG-Native卸载功能；优化自动生成的快捷方式图标。
v1.6，修复切换到32位版时，进程被占用产生的bug。
v1.7，进一步简化安装流程，做到了exe一键安装。
v1.71，修复原版aria2在一键生成快捷方式后由于隐藏后台而导致防火墙没有弹出提示的bug。
v1.72，新增跨文件夹(跨版本)更新功能！你现在可以通过解除限制编译版的管理器给原版更新bt-trackers了，并且原版不会产生任何新服务。只要保持默认的文件夹名称和路径就可以了。
v1.73，进一步优化跨版本更新功能，执行跨版本更新操作后，会自动终止当前进程，并自动启动原版。
v1.8，新增内网穿透功能。注意：内网穿透功能一旦开启，如需关闭，请到aria2管理器中关闭putty守护进程。否则，它会自动断开重连。
aria2为特别优化版，Aria2-Original_edition为原版，只做轻微优化。


解除限制编译(特别优化)版安装说明：
Step1.解压后打开“aria2”文件夹。
Step2.win10用户点击“win8-win10一键安装.exe”,win7用户点击“win7一键安装”。
ps：默认为64位版，32位用户请在安装完成后，点击“切换到32位.exe”。
如需体验完整功能，请点击“她明明超厉害，却过分低调.exe”。

原版（轻微优化）安装说明：
Step1.解压后打开“Aria2-Original_edition”
Step2.直接运行“一键生成快捷方式并启动.exe”，并允许防火墙通过。
如需体验完整功能，请点击“这个管理器不怎么厉害.exe”。
